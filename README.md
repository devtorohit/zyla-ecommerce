# Zyla - Full Stack MERN E-Commerce Platform 🛒

Zyla is a robust, full-featured e-commerce application built using the **MERN stack**. The project focuses on high-level security, seamless user experience, and scalable backend architecture.

## 🚀 Key Features
* **Secure Authentication**: User Login and Signup using **JWT**.
* **Advanced Token Management**: Silent Token Refresh system using Access and Refresh tokens.
* **Protected Routes**: Private routing for the Dashboard and Profile sections.
* **Professional Backend Structure**: Controller-Middleware-Route pattern.

## 🛠️ Tech Stack
* **Frontend**: React.js, Vite, Axios, React Router, CSS3.
* **Backend**: Node.js, Express.js.
* **Database**: MongoDB (Mongoose ODM).
* **Auth**: JWT, Cookie-parser.
